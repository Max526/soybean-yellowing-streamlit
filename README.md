# 大豆葉片黃化偵測與線上輔助診斷平台

本平台使用 YOLO 偵測大豆葉片位置，再於葉片 ROI 中使用 HSV 顏色分析估算黃化比例，最後輸出黃化等級與輔助診斷建議。

## 功能

- 上傳葉片照片
- YOLO 偵測 leaf bbox
- HSV 黃化像素分析
- 顯示原圖、框選圖、黃化 mask
- 顯示黃化比例與診斷等級
- 顯示每個 bbox 的信心分數與黃化比例

## 啟動方式

```bash
pip install -r requirements.txt
streamlit run app.py
```

## 權重設定

預設會嘗試讀取：

```text
weights/best.pt
```

如果權重放在其他位置，可以在網頁側邊欄輸入完整路徑。

## HSV 黃化設定

目前提供三組：

| Preset | HSV 範圍 | 用途 |
|---|---|---|
| Conservative | H 22–35, S 60–255, V 100–255 | 只抓明顯黃化，降低誤判 |
| Balanced | H 18–42, S 40–255, V 80–255 | 預設平衡版本 |
| Loose | H 15–45, S 35–255, V 70–255 | 抓淡黃化，提高召回 |

## 診斷分級

| 黃化比例 | 診斷 |
|---:|---|
| <10% | 正常 / 低黃化 |
| 10–30% | 輕微黃化 |
| 30–60% | 中度黃化 |
| >60% | 嚴重黃化 |

## 注意

本平台為輔助診斷工具，HSV 門檻會受到光源、相機、背景影響，需搭配實際田間觀察與專業判斷。
